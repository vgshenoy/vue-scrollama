# vue-scrollama workspace

Monorepo layout:

- `packages/vue-scrollama` - publishable library package
- `apps/example` - local example app wired to the workspace package

Common commands:

- `pnpm install`
- `pnpm run test`
- `pnpm run lint`
- `pnpm run typecheck`
- `pnpm run build`
- `pnpm run dev:demo`
